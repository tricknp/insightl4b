Consultando resultado de banca de direção no Detran-DF com Python

13 September 2016 on getran, detran, python, detran-df, scripting

Esses dias eu estava fazendo algumas pesquisas de segurança e levantamentos de vulnerabilidades em aplicativos mobile, e acabei me deparando com o aplicativo do Detran-DF. Pesquisando mais a fundo e fazendo alguns testes, acabei descobrindo alguns Web services e API's que são utilizados pelo aplicativo do Detran na consulta dos dados na base de dados do Getran(Sistema de gestão de transito).

Dados - API:


Com uma das API na mão é possível realizar consultas na base de dados da banca de direção do Detran-DF utilizando apenas o CPF. A api irá retornar alguns dados como: Nome completo, CPF, renach, Data de nascimento, CFC, Data e horario do seu exame prático.

Modo de uso: ./getran.py < cpf > | ./getran.py 1234567890

Output:
Nome completo: JHONATHAN DAVI * * * * * *
CPF: * * * .* * * .* * * -**
Renach: * * * * * *
Data de nascimento: * * / * * / * * * *
CFC: CFC AB SERRANA CITY
Data: 16/01/2016 Hora: 06:00
Resultado: APROVADO

Código: https://github.com/jh00nbr/Python/blob/master/getran_df.py
