Phishruffus - Intelligent threat hunter and phishing servers

27 October 2017

O Phishruffus é uma ferramenta projetada para a identificação de ameaças e servidores DNS maliciosos na internet que são utilizados para a prática ilícita de Phishing bancários.

A ferramenta trabalha realizando consultas DNS em dois domínios bancários (Banco do Brasil e Caixa) e através de white-lists com ips existentes no código é possível realizar a validação se o domínio está há responder para o servidor original do banco.

Executando ./phishruffus.py --listdns dns_servers.txt --timeout 5

Github https://github.com/jh00nbr/Phishruffus
